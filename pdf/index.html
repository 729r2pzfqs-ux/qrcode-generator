FAQ

Where should I host my PDF?

Google Drive (free), Dropbox, OneDrive, or your own website all work. Make sure the link is publicly accessible (anyone with link can view).

Can I update the PDF later?

Yes! If you replace the file at the same URL, the QR code will show the new version. With Google Drive, just upload a new version of the same file.

Is there a file size limit?

The QR code just links to your PDF ‚Äî there's no limit. However, smaller PDFs (under 10MB) load faster on mobile devices.
